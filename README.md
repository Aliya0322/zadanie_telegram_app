# Class App - Образовательная платформа

Приложение для управления группами студентов и домашними заданиями, разработанное для Telegram WebApp.

## Технологии

- **React 19** - UI библиотека
- **TypeScript** - типизация
- **Vite** - сборщик и dev-сервер
- **Tailwind CSS** - стилизация
- **Konsta** - UI компоненты для мобильных приложений
- **React Router** - маршрутизация
- **Axios** - HTTP клиент
- **Heroicons** - иконки

## Архитектура проекта

Проект разделен на два репозитория:

- **Frontend** (текущий репозиторий) - React приложение для Telegram WebApp
- **Backend** (отдельный репозиторий) - API сервер

Подробная информация о бэкенде и API endpoints находится в файле [BACKEND.md](./BACKEND.md).

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

3. Настройте переменные окружения в `.env`:
```
VITE_API_BASE_URL=http://localhost:3000/api
```

4. Запустите dev-сервер:
```bash
npm run dev
```

5. Соберите проект для продакшена:
```bash
npm run build
```

## Основные функции

### Аутентификация
- Вход как учитель или студент
- Интеграция с Telegram WebApp SDK
- Управление сессиями через localStorage

### Группы
- Создание и управление группами
- Просмотр списка групп
- Детальная информация о группе
- Добавление/удаление студентов

### Домашние задания
- Создание домашних заданий
- Просмотр заданий по группам
- Отправка выполненных заданий (для студентов)
- Оценка заданий (для учителей)

### Расписание
- Просмотр расписания занятий
- Напоминания о предстоящих занятиях

## Разработка

Проект использует:
- **TypeScript** для типобезопасности
- **ESLint** для проверки кода
- **Tailwind CSS** для стилизации
- **Konsta** для нативных мобильных компонентов

## Telegram WebApp

Приложение оптимизировано для работы в Telegram WebApp. При запуске вне Telegram SDK автоматически определяется режим разработки.
